CTRL + SHIFT + UP / DOWN = ПЕРЕМЕЩАЕТ СТРОЧКУ КОДА ВВЕРХ ВНИЗ 
СTRL + SHIFT + I = СПРАВКА ПО ФУНКЦИИ  
python -m venv venv = установить  виртуальное окружение в текущую папку  с названием venv(названием папки устанавливаемой)
rm .\venv\ =  удалить виртуальное  окружение
.\venv\Scripts\activate = активировать виртуальное окружение
deactivate = деактивировать виртуальное окружение
pip list = список пакетов установленых
